
# Text Seach Engine .

This is a small project for simple search engine .

We will extract keywords automatically based on well-known technique TF.IDF ,

and synonym clusters are labeled by users .

## Technique :

	1. Database Schema .
	2. Web UI .
	3. Dijango framework .
	4. Uploading .
	5. Words splitting . 
	6. Indexing .
	7. SQL join .
	8. Rating (optional) .
